# AI Resume Analyzer

This is a Python-based project that analyzes how well a resume matches a job description using basic NLP techniques.

## 🚀 Features
- Calculates resume match score
- Compares resume with job description
- Provides improvement suggestions
- Beginner-friendly AI project

## 🛠️ Technologies Used
- Python
- Scikit-learn
- NLP (CountVectorizer, Cosine Similarity)

## ▶️ How to Run
1. Install required libraries:
   pip install scikit-learn

2. Run the program:
   python app.py

3. Paste your resume and job description when prompted

## 💡 Output
- Displays match score in percentage
- Provides suggestions to improve resume
